# Smart Shopping Assistant Bot

Telegram-бот для сохранения покупок, анализа расходов и финансового планирования.

## Возможности

- Добавление покупок вручную или в "умном" формате
- Выбор категории через кнопки
- Список покупок и фильтрация по категории
- Сумма расходов по категориям и общая сумма
- Удаление и очистка записей
- Экспорт списка покупок в текстовый файл
- Базовая статистика расходов

## Установка

1. Установите Python 3.10+
2. Создайте виртуальное окружение:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
3. Установите зависимости:
   ```powershell
   pip install -r requirements.txt
   ```
4. Скопируйте файл `.env.example` в `.env` и вставьте токен бота:
   ```powershell
   Copy-Item .env.example .env
   ```
5. Откройте файл `.env` и замените `your_bot_token_here` на токен Telegram-бота.
6. Запустите бота:
   ```powershell
   python main.py
   ```

## Команды

- /start — приветствие и инструкции
- /add — начать добавление покупки
- /list — показать все покупки
- /list <category> — показать покупки по категории
- /total — общая сумма
- /total <category> — сумма по категории
- /delete <id> — удалить запись
- /clear — очистить данные с подтверждением
- /export — экспортировать покупки в текстовый файл
- /stats — статистика расходов

## Примечание

Для Windows используйте PowerShell и путь `.venv\Scripts\Activate.ps1`.
